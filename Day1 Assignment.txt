Q2.What are the JavaScript Engines?
A JavaScript engine is a Software Components that execute JS code. The first JS engine is  interpreters, but all relevant modern engines use just-in-time compilation
 for improved performance. 
JavaScript engines are typically developed by web browser vendors, and every major browser has one browser, the JavaScript engine runs with Document Object Model.
example-  
	v8 engine for chrome in 2008
	Apple developed Nitro engine for Safari browser
	SpiderMonkey for Firefox
and others browser have a own engines.
	

Q3.    Why We use JavaScript?
JavaScript is a scripting language used to develop web pages. Developed in Netscape, JS allows developers to create a dynamic and interactive web page 
to interact with visitors and execute complex actions. It also enables users to load content into a document without reloading the entire page

Q4.    What is JavaScript? And also Explain the History of JavaScript?
JavaScript is a light-weight object-oriented programming language that is used by several websites for scripting the webpages. It is an interpreted,
 programming language. JavaScript enables dynamic interactivity on websites when it is applied to an HTML document.
JS helps the users to build modern web applications to interact directly without reloading the page every time. JavaScript is commonly used to dynamically 
modify HTML and CSS  files.

History of JS 
Java Script was invented by Brendan Eich in 1995. It was developed for Netscape2 and become ECMA-262 standard in 1997.
